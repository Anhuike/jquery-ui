jQuery UI Carousel

Example of jQuery UI Carousel using default options and UI Darkness theme.

Current widget options:

  • scroll - How many carousel items to scroll on each event.
  • visible - How many carousel items are visible at one time.
  • itemWidth - The width of the carousel items can be set if not controlled by CSS.
  • setWidth - The width of the clip div will be calculated based on the width of the carousel item and visible items.
  • track - Whether the tracking element will be visible, displaying the current position of the carousel.
  • easing - The easing applied to the carousel. See http://jqueryui.com/demos/effect/#easing
  • speed - The speed at which the carousel animates.

The carousel requires jQuery, jquery.ui.core.js and jquery.ui.widget.js. jquery.effects.core.js is required for additional easing options.

This widget requires jquery.ui.carousel.js and CSS rules outlined in jquery.ui.carousel.css.
